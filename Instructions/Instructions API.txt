If you don't have a API key:

- Visit: https://www.renewables.ninja
- Sign up for free and register with your e-mail address
- Confirm your account on e-mail address
- Log in and click on Profile
- Copy Token (your API key) and paste it in ninja_api_key or generate new API token

How to add your Renewables.ninja API key:

1. Open the file config.json (C:...\AppData\Roaming\GREENADVISE).

2. Replace empty input "" with your real API key, for example:

   {"ninja_api_key": "123abcYourKeyHere"}


3. Save the file. Restart GREENADVISE.
